## CPC_BACKEND

createTable.js

childTable과 descendants 테이블을 생성하여 sql에 데이터 삽입

elastic.js

elasticsearch api를 본 시스템의 nodejs에서 맞춰 활용하기 위해 커스텀하여 재 api화

indexing.js

elasticsearch에 데이터 인덱싱

initIndex.js

elasticsearch에 애널라이저 세팅 매핑

postprocess.js

서버를 시작한 후 api에 들어가는 module

preprocess.js

본 서버를 시작하기 전에 구동하는 작업
